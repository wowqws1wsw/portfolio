What is HTML, PHP, CSS, Vue and JS?

HTML

HyperText Markup Language ‚Äî struktur dasar halaman web.

PHP

Bahasa sisi-server untuk membuat konten dinamis dan berinteraksi dengan database.

CSS

Digunakan untuk mengatur tampilan (layout, warna, tipografi) halaman web.

JavaScript

Bahasa pemrograman yang membuat halaman menjadi interaktif.

React.js

library JavaScript open-source yang diciptakan oleh Facebook untuk membangun antarmuka pengguna (UI) yang interaktif dan dinamis pada aplikasi web atau mobile.

Vue

Framework progresif JavaScript untuk membangun antarmuka pengguna.
